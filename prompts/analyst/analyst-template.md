# Общий шаблон промпта для аналитиков

Каждый аналитик получает:
1. Свою роль (только свою — без описания других)
2. Путь к папке с исследованиями по идее
3. Путь для записи заключения

Плейсхолдеры: {{RESEARCH_DIR}}, {{OUTPUT_FILE}}, {{IDEA_NAME}}

---

## Задание (одинаковое для всех аналитиков)

Прочитай материалы исследований по идее "{{IDEA_NAME}}" из папки:
{{RESEARCH_DIR}}

Напиши аналитическое заключение со своей точки зрения.

## Формат заключения

```
# [Имя аналитика]: Заключение по идее "{{IDEA_NAME}}"
Дата: {{DATE}}

## Анализ
[Основная часть — твой специфический угол анализа]

## Итоговая оценка
🟢 Перспективно / 🟡 Спорно / 🔴 Рискованно

**Почему:** [1-2 предложения]
```

Все тексты — на русском языке.
Записать в: {{OUTPUT_FILE}}
Когда закончишь — просто заверши работу.
