# prompts/ — Lean-промпты для агентов

Эта папка содержит оптимизированные промпты для всех агентов Research Lab.

## Проблема, которую решаем

Раньше в `task` агента улетал весь файл `config/creative-team.md` (870 строк) —
вместо ~60 строк, которые реально нужны одному агенту.
Умножаем на 12 креативщиков → миллионы лишних токенов.

Теперь каждый агент получает **только свою роль** + задание + путь к файлу.

## Структура

```
prompts/
├── creative/       ← 12 креативщиков
├── jury/           ← 10 членов жюри
├── research/       ← 16 исследователей
├── analyst/        ← 10 аналитиков
└── operations/     ← Дедупликатор, Считовод, Чистильщик, Верховный аналитик
```

## Плейсхолдеры

Перед запуском агента Crabe (оркестратор) заменяет плейсхолдеры на реальные пути:

| Плейсхолдер | Что подставить |
|---|---|
| `{{COUNT}}` | Количество идей для генерации (например: 20) |
| `{{OUTPUT_FILE}}` | Полный путь к выходному файлу |
| `{{IDEAS_FILE}}` | Путь к файлу со списком идей (для жюри) |
| `{{IDEAS_DIR}}` | Путь к папке с идеями (для Дедупликатора) |
| `{{SCORES_DIR}}` | Путь к папке с оценками (для Считовода) |
| `{{IDEA_FILE}}` | Путь к файлу с одной идеей (для исследователей) |
| `{{IDEA_NAME}}` | Название идеи (для аналитиков) |
| `{{RESEARCH_DIR}}` | Папка с исследованиями (для аналитиков) |
| `{{DATE}}` | Текущая дата YYYY-MM-DD |

## Как использовать

```python
# Пример: запуск Прагматика
prompt = open("prompts/creative/creator-01-pragmatik.md").read()
prompt = prompt.replace("{{COUNT}}", "20")
prompt = prompt.replace("{{OUTPUT_FILE}}", f"{run_dir}/01-ideas/raw/creator-01-pragmatik-ideas.md")

sessions_spawn(
    mode="run",
    task=prompt
)
```

## Примерный размер промпта

| Тип агента | Раньше | Сейчас |
|---|---|---|
| Креативщик | ~5-10K токенов | ~600-800 токенов |
| Член жюри | ~4-8K токенов | ~700-900 токенов |
| Исследователь | ~3-6K токенов | ~500-700 токенов |
| Аналитик | ~5-10K токенов | ~600-900 токенов |

Экономия на одном прогоне (55 агентов): **~80-90% токенов на промптах**.
