# 🔬 Research Lab — Поиск бизнес-идеи

## Цель
Найти идею цифрового бизнеса, который можно запустить в одиночку с помощью AI-агентов.

**Целевые показатели:**
- Минимальные начальные вложения
- Быстрая первая прибыль
- $100K/мес чистой прибыли через год

**Что есть:** компания в США, банковский счёт, Stripe, опыт запуска бизнесов (офлайн).

## Pipeline (порядок работы)

1. **Креативный отдел** (10 агентов) → генерируют по 20 идей каждый → 200 идей
2. **Дедупликатор** → убирает повторы, объединяет похожие
3. **Жюри** (10 агентов) → оценивают каждую идею по 7 критериям (1-10)
4. **Считовод** → считает средние баллы
5. **Менеджер** → отбирает топ-20
6. **Исследователи** (10 агентов) → глубокий поиск по каждой из 20 идей
7. **Чистильщик** → убирает мусор и битые ссылки
8. **Аналитики** (5 агентов) → пишут заключения
9. **Верховный аналитик** → консолидирует отчёты
10. **Презентация** → итоговые резюме для Сергея

## Критерии оценки (7 шт.)

1. **Лёгкость запуска** — можно ли начать одному, без больших вложений
2. **Скорость до первой прибыли** — как быстро первый доллар
3. **Потенциал масштабирования** — реально ли $100K/мес
4. **Лёгкость привлечения клиентов** — понятно ли, где брать покупателей
5. **Объём рынка** — достаточно ли большой
6. **Автоматизируемость** — можно ли вести на AI-агентах без найма
7. **Конкурентное преимущество** — есть ли шанс выделиться

## Служебные роли

- **Архивариус** — фиксирует всё в файлы
- **Дедупликатор** — убирает повторы идей
- **Считовод** — считает средние оценки
- **Менеджер** — отбирает топ-20
- **Чистильщик** — убирает мусор из исследований
- **Верховный аналитик** — консолидирует отчёты
- **Координатор** (Crabe) — управляет pipeline
